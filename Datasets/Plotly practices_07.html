Tile Map Layers in JavaScript

The map is composed of various layers, of three different types:

  • layout.map.style defines the lowest layers, also known as your "base map"
  • The various traces in data are by default rendered above the base map (although this can be controlled via the below attribute).
  • layout.map.layers is an array that defines more layers that are by default rendered above the traces in data (although this can also be controlled via the below attribute).

  1. A Simple Example Map rendered with "open-street-map" tiles.
    • Use unpack(rows,key) function to get cell values from csv file column with key as header
    • use map: { style: "open-street-map",}
    • use dragemode options
  2. Replot the ISHR Speaker Distribution using the OpenStreetMap tiles
    • Retrieve data from an online csv file
    • Use dragmode:"zoom" (default)
    • Use Base Maps in layout.mapbox.style Options
    • Use hovertemplate to customize label display: %{marker.size}
  3. Replot ISHR Speaker Map using Base Tiles from the USGS
    • uses a public USGS imagery map, specified in layout.map.layers, and which is rendered below the data layer.
    • style option "white-bg"
    • Use layers.sourcetype: "raster"
  4. Base Tiles from the USGS, radar overlay from Environment Canada

  5. Creating map over dark tiles.
    • Retrieve data from an online geojson file
    • Use Geojson.io to view the json file in map and table format
    • check earthquake.json to understand the data structures
      • "features"->"geometry"->"coordinates[0]": longitude value of each earthquake location
      • "features"->"geometry"->"coordinates[1]": latitude value of each earthquake location
      • "features"->"properties"->"mag": magnitude of each earthquake
